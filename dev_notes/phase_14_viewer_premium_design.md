# Phase 14: Project Viewer & Card Premium Design Update

## 개요
이전에 구현한 프로젝트 모달 뷰어(`ProjectDetailsViewer.tsx`)와 크리에이터 대시보드의 카드(`ProjectCard.tsx`)의 디자인 및 인터랙션을 한 단계 업그레이드하여, 프리미엄 포트폴리오 사이트 수준의 체감 효과를 구현했습니다.

## 주요 작업 내역
1. **애니메이션 엔진 정상화**
   - 기존에 삽입되었으나 작동하지 않던 `tailwindcss-animate` 플러그인을 `package.json` 및 `tailwind.config.ts`에 정상적으로 설치 및 등록 완료.
   - 이를 통해 아래에 서술된 모든 마이크로 애니메이션들이 완벽하게 동작하게 됨.

2. **포트폴리오 카드(`ProjectCard.tsx`) 인터랙션 고도화**
   - 카드 호버 시 전체적인 카드가 살짝 떠오르는(`hover:-translate-y-1`) 부유 효과와 깊은 그림자(`shadow-[0_20px_40px_-10px_rgba(0,0,0,0.15)]`) 적용.
   - 내부 이미지 썸네일이 서서히 확대되는(`scale-[1.03]`) 패럴랙스 느낌의 모던 줌 효과 적용.
   - 카드 하단의 타이틀/카테고리 텍스트 영역이 호버 시에만 하단에서 부드럽게 솟아오르는(`translate-y-4` -> `translate-y-0`) 인터랙티브 UI 추가.

3. **모달 진입 애니메이션(`ProjectModal.tsx`)**
   - 기존의 단순 슬라이드-업을 넘어서, 바탕화면이 블러 처리(`backdrop-blur-md`)되며 어두워짐과 동시에 모달 창이 화면 중앙에서 살짝 커지면서(`zoom-in-[0.97]`) 나타나는 Apple/Behance 스타일의 프리미엄 진입 효과 추가.
   - X 닫기 버튼에도 블러 효과(`bg-black/40 backdrop-blur-md`) 및 hover시 스케일 업 애니메이션을 더해 작은 디테일까지 완성도 향상.

4. **뷰어 내부 시각적 최적화 (`ProjectDetailsViewer.tsx`)**
   - `prose-neutral` 톤의 본문 텍스트와 최적의 줄간격/자간 설정 적용 완료.
   - 커버와 본문 사이에 둥글고 부드러운 분리선 추가.

## 식별 및 해결된 버그
- **텍스트 정렬 오류**: 에디터에서 개별적으로 지정한 정렬이 뷰어에서 렌더링될 때 `prose` 컨테이너 기본값의 부재로 인해 강제로 가운데 정렬되는 현상을 수정.
- **애니메이션 미작동 문제**: CSS 유틸리티 클래스가 정상 컴파일되지 않던 이슈를 플러그인 추가로 해결.
