# 크리에이터 포트폴리오 뷰어 컴포넌트 세팅 지침

## 진행 사항
- `src/app/creator/[creator_name]/page.tsx` 라우트를 생성하고 Behance 갤러리 UI를 모방한 뼈대를 기획합니다.

## 개발 권고/지침 사항
1. **SSG / ISR 렌더링 고려**: 각 크리에이터의 포트폴리오 페이지는 SEO와 초기 로딩 속도 최적화를 위해 매우 중요하므로 SSR 혹은 SSG를 지향합니다.
2. **동적 위젯 렌더링**: 사용자가 배치한 위젯의 타입(이미지, 영상, Before/After)에 따라 순차적으로 렌더링될 수 있도록 컴포넌트화를 철저히 분리합니다. `Array.map`을 통해 렌더링하게 됩니다.
3. **디자인 테마**: 전체 화면 너비보다는 콘텐츠에 집중할 수 있도록 가운데 정렬 형태(예: `max-w-6xl mx-auto`)의 컨테이너를 사용합니다.
