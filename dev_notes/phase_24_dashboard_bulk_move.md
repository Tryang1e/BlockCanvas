# Phase 24: 크리에이터 대시보드 게시물 관리 사용성 개선 (Bulk Move & Section Reorder)

### 1. 개요
기존 대시보드에서는 섹션 간 게시물 이동을 위해 드래그 앤 드롭 방식을 사용하였으나, 데이터가 많아질 경우 화면 밖으로 드래그해야 하는 불편함이 컸음. 이를 해소하기 위해 UI/UX를 전면 개선함.

### 2. 주요 작업 내용
- **게시물 일괄 이동 (Bulk Move) 추가**
  - 각 게시글 행에 체크박스를 추가하고, `selectedProjectIds` 상태로 다중 선택 지원.
  - 선택된 게시글이 있을 경우 화면 상단에 플로팅 바(Sticky Bar) 표시.
  - 드롭다운에서 목표 섹션을 선택하고 '이동' 버튼 클릭 시 드래그 없이 다중 데이터를 즉시 이동.
  - 이동 후 `updateProjectOrderAction`을 호출하여 DB와 동기화.

- **섹션 순서 변경 모달 (Section Reorder Modal) 연동**
  - 무거운 섹션 컨테이너 통째로 드래그하는 방식을 제거.
  - 대시보드 상단에 `[⚙️ 섹션 순서 관리]` 버튼을 추가.
  - 클릭 시 `SectionReorderModal` 팝업을 띄워, 컴팩트한 리스트에서 섹션 순서만 가볍게 정렬 가능.
  - 모달 뒤에는 `fixed inset-0 bg-black/60` 배경을 추가하여 기존의 렌더링 겹침 버그(버튼 안 눌림 등) 수정.

### 3. 해결한 문제점
- Next.js 컴파일 오류 및 무한 루프 가능성 점검.
- Modal 백드롭 누락으로 인한 페이지 클릭 불가/스크롤 막힘 현상 완벽 조치.
- PowerShell 사용 시 한글 깨짐 현상을 피하기 위해 파일 API를 직접 사용하여 작성.
