# BlockCanvas(블록캔버스) 사용성 평가 문답지 (Usability Questionnaire)

본 설문은 마인크래프트 크리에이터 전용 포트폴리오 플랫폼 **'BlockCanvas(블록캔버스)'**를 직접 사용해 보신 생생한 경험과 후기를 바탕으로, 크리에이터 분들께 더 나은 가치와 고도화된 기능들을 제공해 드리기 위해 기획되었습니다. 

작성해 주신 정성 어린 문답은 향후 서비스 개발 방향을 설정하고 사용자 경험(UX)을 극대화하기 위한 가장 중요한 이정표로 활용됩니다. 더욱 완벽하고 매력적인 플랫폼으로 발전할 수 있도록 소중한 피드백을 보태주시면 대단히 감사하겠습니다.
